Common Questions

1. How do I design a batch distillation column?

Answer: Distillation Designer currently supports the McCabe Thiele Method which is only applicable for binary continuous distillation designs.

2. Is this program useful for real designs of distillation columns?

Answer: The McCabe Thiele method is used mostly in an education context. You will need to decide for yourself if the method is applicable for your own purposes.

3. Where can I get other VLE data files?

A web page has been setup to allow you to download VLE datafiles other users have created. It is hoped if you will send us any useful datafiles you produce for others to benefit from.

4. What's the format of the VLE datafiles.

The VLE datafile is of a simple readable format, more information can be found here.
