Format of the VLE Datafile

  • Data on the same line is separated by <Tabs>.
  • Order of the data is fixed.
  • Data titles are required and correctly entered.
  • Molecular Weight, Latent Heat, Specific Heat Capacity and Pressure are optional, but must be defined as 'Not Set' if no data can be entered.
  • For VLE Data (x (mol frac),y (mol frac),temperature (degreesC) ):
    First line will hold the number of data rows below.
    Data composition must start at

    0.0<tab>0.0<tab>Boiling Point of Component 1

    and end with

    1.0<tab>1.0<tab>Boiling Point of Component 2
