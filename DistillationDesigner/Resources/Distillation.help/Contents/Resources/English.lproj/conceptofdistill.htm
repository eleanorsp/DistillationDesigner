Concept of Distillation Designer

Distillation Designer is split into 3 sections (or design steps):

Composition

Define and review the Liquid/Vapour/Boiling point data.

Specification

  1. The VLE data is transposed into the McCabe Thiele (Vapour/Liquid composition) diagram.
  2. Define the design requirements and information which the feed will enter the distillation column.
  3. Identify the minimum reflux ratio (Rmin)
  4. Set the 'optimum reflux factor' to present an optimum result.
  5. Then plot your McCabe Thiele diagram with the 'optimum' reflux to get the actual theorectical stages for that reflux ratio.

Optimisation

An optimal design for a binary component distillation column suggests a reflux factor of 1.2-1.4Rmin. Investigate how a range reflux ratios affect the number of theorectical stages in the design.

Once you decide of an optimal reflux factor return to the Specification tab to calculate your final design.
